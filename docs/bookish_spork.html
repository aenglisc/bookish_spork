    stub request with fun or particular response

    Response can be* either fun((bookish_spork_request:t()) -> bookish_spork_response:response())

    • or response data structure bookish_spork_response:response()

    Example:

      bookish_spork:stub_request(fun(Request) ->
      case bookish_spork_request:uri(Request) of
          "/bookish/spork" ->
              [200, [], <<"Hello">>];
          "/admin/sporks" ->
              [403, [], <<"It is not possible here">>]
      end
  end, _Times = 20)
